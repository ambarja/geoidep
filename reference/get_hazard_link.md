# Retrieve the links of the SIGRID for information on Disaster Risk Management.

Retrieve the links of the SIGRID for information on Disaster Risk
Management.

## Usage

``` r
get_hazard_link(type = NULL)
```

## Arguments

- type:

  A string. Select only one from the list of available layers, for more
  information please use `get_data_sources(provider = "INAIGEM")`.
  Defaults to NULL.

## Value

A string containing the URL of the requested file.
