# Retrieve the links to MTC for information on transport and telecomunication.

Retrieve the links to MTC for information on transport and
telecomunication.

## Usage

``` r
get_mtc_link(type = NULL)
```

## Arguments

- type:

  A string. Select only one from the list of available layers, for more
  information please use `get_data_sources(provider = "MTC")`. Defaults
  to NULL.

## Value

A string containing the URL of the requested file.
