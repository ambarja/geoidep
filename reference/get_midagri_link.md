# MIDAGRI links for obtaining cartographic information

MIDAGRI links for obtaining cartographic information

## Usage

``` r
get_midagri_link(type = NULL)
```

## Arguments

- type:

  A string. Select only one of the following layers; 'vegetation cover',
  'agriculture sector', 'oil palm'. Defaults to NULL.

## Value

A string containing the URL of the requested file.
